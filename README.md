# CropCare - VLM-based potato leaf disease assistant
This repository contains developmet of adapting a Vision-language model fine-tuning for potato leaf disease diagnosis and conversational agricultural support. The project combines image understanding and natural language interactions to support agronomists with on-field detection and diagnosis of diseases for faster decision-making in crop health assessment. 

## Project Overview

The goal of this work is to fine-tune a vision-language model for potato leaf disease diagnosis using curated image-text data and parameter-efficient training methods. The system is designed to identify common potato leaf symptoms and provide conversational responses that may help users interpret plant health conditions in a practical way.
